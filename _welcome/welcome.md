# Welcome

Hello and welcome to this small corner of the internet. I'm planning to start posting here more regularly again, and I'll be updating older content and project pages as time allows.

This site is very much a work in progress — a place to collect ideas, experiments, and spare parts that don’t quite fit into any of the other outlets where I write or develop code. Expect the content to wander a bit: some days it might be modern Unity development, other days it might be Sega Master System homebrew, or even small discoveries I make while tinkering with CSS and web layouts.

Content is grouped into categories and listed by date. There’s no search feature yet, but as the site grows I’ll add one — and probably write about the process too. My goal is to build as much of this site myself as possible, without relying on frameworks or static site generators.

Hopefully you find something here interesting or useful. All of the code and related material will also be available on GitHub for anyone who wants to dig deeper.
